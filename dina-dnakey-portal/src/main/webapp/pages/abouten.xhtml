Using the DNA key search tools one can perform similarity searches between an unknown DNA sequence (so-called ) and a large number of sequences stored in a several . The search method used here is , a statistical method for quickly finding sequence regions, or , with high similarity. The results of a search are displayed as a list of the top 10 hits in the selected database for each query sequence. Currently there is a limitation in the maximum number of sequences (100) that can be used for an individual search. If the search finds sequences in the database with sufficiently high (≥99%), then the results are displayed, with the data provided on the sequence from the database as a table with a number of clickable links. Depending on the database that is used, some of the columns have no information for the displayed hit sequences. Data fields, in the order of display, are catalog number in the (), the scientific name with link to (), the name of the gene region represented by the hit sequence (), accession number and link to the sequence in (), ID number and link to the sequence in (), percentage similarity (), score (), as well as E-value (). The last three columns are the details related to the algorithm. If one expands the row for the hit sequence (by clicking on the arrow symbol on the left side), and the hit sequence has links to data in other databases, additional information will be displayed. Among other things, information from http://www.naturarv.se/ will be shown (Collection, Swedish names, Local, Coordinates, Date, Collector). In addition, the “Map” button shows location coordinates on Google Maps The “Images” button displays clickable images stored in “Naturarv” (right click on the picture shows more links to additional databases). Finally, the button shows a graphical comparison (alignment) between positions in the parts where the query sequence ("Query") and the hit sequence ("Subjc") have high similarity. If no match is made between the sequences in the database and query sequence - which can happen, for instance, when the query sequence represents a completely different gene than is currently represented in the database - then a link is displayed with the following text “Can not match any records in the selected database! Try a search of GenBank.” If you click on “Search GenBank” a new BLAST search (with default settings) is performed on GenBank's BLAST portal, and then a new link to the results is generated.

FASTA format
 A standardized format for gene sequence data. The information about each sequence consists of an information line that begins with ">", followed by the DNA sequence on one or more subsequent lines. Currently the DNA key only supports using a DNA sequence as a search criterion. Examples of DNA sequences in FASTA format can be viewed by using the auto-paste function for test sequences into the DNA key search field (tab "Testing the System"). See also Wikipedia link.
